# Customer Churn Prediction

## Project Overview
Built a machine learning model to predict which customers are likely to leave a telecom company (churn).

## Key Highlights
- Analyzed a real-world Telco Customer Churn dataset (7043 records)
- Performed Exploratory Data Analysis to understand churn patterns
- Created an automated preprocessing pipeline using **scikit-learn Pipeline + ColumnTransformer**
- Trained a RandomForestClassifier model with **79.1% accuracy**
- Focused on business insight: identifying customers at risk of churning

## Visualizations Included
- Churn distribution
- Tenure distribution by churn status

## Tools Used
- Python, pandas
- scikit-learn (Pipeline, ColumnTransformer, RandomForestClassifier)
- Seaborn & Matplotlib (for visualizations)

## How to Run
1. Download the dataset `WA_Fn-UseC_-Telco-Customer-Churn.csv`
2. Open `customer_churn_analysis.ipynb`
3. Run all cells

## Learning Outcomes
- Handled categorical and numeric features properly
- Understood the importance of recall in imbalanced classification problems (churn prediction)
- Built a clean, reusable end-to-end pipeline
