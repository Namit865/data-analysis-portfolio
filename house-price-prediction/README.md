# House Price Prediction

**Simple End-to-End Project**

- Cleaned messy house prices dataset (81 columns with lots of missing values)
- Created smart binary features (`Has_Pool`, `Has_Fireplace`, `Has_Garage`)
- Used Pipeline + ColumnTransformer for clean preprocessing
- Built RandomForest model with R² = 0.76

**Tools:** pandas, scikit-learn, Seaborn, Matplotlib

Visualization: Actual vs Predicted house prices included in the notebook.
